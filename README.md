## About project

Project for creating motivational posters with fully functional API. Project has been developed in Laravel framework.

- Images API  - list all images, upload new images, delete images
- Posters API - list all posters with images belongs to that poster, store, update and delete posters
- Albums API  - list all albums with posters, store, update and deleting albums
